## 🎯 学习目标

完成第一阶段后，你应该能够：

1. **熟练使用 ES6+ 语法**
2. **理解 JavaScript 作用域机制**
3. **掌握异步编程模式**
4. **灵活运用数组高阶函数**
5. **具备 Node.js 开发基础**

## 📝 学习建议

1. **理论与实践结合**: 先阅读代码注释，然后运行代码观察结果
2. **动手修改**: 尝试修改示例代码，观察不同效果
3. **对比学习**: 注意 Python 和 JavaScript 的差异
4. **记录笔记**: 记录重要的概念和易错点
5. **及时复习**: 每天复习前一天的内容

## 🔧 运行代码

在 VS Code 中：
1. 打开终端 (Ctrl + `)
2. 导航到文件所在目录
3. 运行 `node 文件名.js`

例如：
```bash
cd "c:\Users\lenovo\Desktop\基础\第一阶段：现代 JS 核心（基础补强）\1.变量与作用域"
node 变量与作用域基础.js
```
