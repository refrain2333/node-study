# 第一阶段：现代 JS 核心（基础补强）

> **学习目标**：掌握 ES6+ 核心语法，为 Node.js 开发打下坚实基础
> **学习时间**：建议 3 天
> **学习方式**：理论 + 实践 + 对比

## 📁 文件结构

```
第一阶段：现代 JS 核心（基础补强）/
├── 1.变量与作用域/           # let, const, 块级作用域
├── 2.箭头函数/              # 箭头函数语法和使用场景
├── 3.异步编程/              # Promise, async/await
├── 4.数组方法/              # map, filter, find, reduce
└── 5.综合练习/              # 综合应用和实战项目
```

## 🎯 学习重点

### 1. 变量与作用域
- ✅ 理解 `let`, `const` 与 `var` 的区别
- ✅ 掌握块级作用域的概念
- ✅ 避免 Python 习惯带来的全局污染

### 2. 箭头函数
- ✅ 掌握 `(arg) => { ... }` 语法
- ✅ 理解 `this` 绑定的差异
- ✅ 适用于 Node.js 中间件开发

### 3. 异步编程
- ✅ 理解 Promise 的 `then/catch` 机制
- ✅ 掌握 `async/await` 语法糖
- ✅ 能够处理异步错误

### 4. 数组方法
- ✅ 熟练使用 `.map()`, `.filter()`, `.find()`
- ✅ 理解函数式编程思想
- ✅ 对比 Python 列表推导式

## 📖 学习建议

1. **每天学习 2-3 个主题**
2. **每个示例都要亲手运行**
3. **对比 Python 和 JavaScript 的差异**
4. **完成综合练习巩固所学**

## 🚀 开始学习

从 `1.变量与作用域/` 开始，按照文件夹顺序逐步学习。
